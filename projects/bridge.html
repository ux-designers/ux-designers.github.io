Bridge - Webapp

How to have everything you need at one place.

Bridge is a browser-based dashboard which therefore runs on all operating systems. It allows to have quick access to relevant daily information for each employee.

The dasboard collects all kinds of apps, which support your daily work, in a place.

The Activity Recording App (CAT2) is an example out of the dashboard widgets.

With the visual help of the task bar it is easy to distribute tasks over a day. The specific color coding for each task and the favorites list make it even more easier.
